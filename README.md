# 🎓 Student Marks Prediction using Linear Regression

## 📌 Overview

This project uses **Linear Regression** to predict student marks based on study hours. It demonstrates a complete yet simple machine learning workflow, helping beginners understand how models are trained and used for real-world predictions.

The goal is to build a system where users can input study hours and receive predicted marks based on learned patterns.

---

## 🚀 Features

* Predict marks from study hours input
* Built using `scikit-learn`
* Simple and beginner-friendly dataset
* Real-time prediction system
* Model evaluation using MSE and R²

---

## 🛠️ Tech Stack

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn

---

## 📂 Workflow

1. Create dataset (hours vs marks)
2. Visualize data (scatter plot)
3. Prepare input (X) and output (y)
4. Train Linear Regression model
5. Evaluate performance
6. Predict marks using user input

---

## 💻 Example

```bash
Enter study hours: 6  
Predicted Marks: 48
```
